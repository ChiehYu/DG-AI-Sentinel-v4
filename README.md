# 🛡️ DG AI Sentinel v4.0 旗艦多螢幕資訊戰情室與 200 萬信貸實戰系統

> [!IMPORTANT]

## 🌟 一、 V4.0 旗艦架構與核心升級亮點 (Architecture & Innovations)

### 1. 專業多螢幕單頁面戰情室佈局 (`Single-Screen War Room Grid`)
為了讓使用者能在單一畫面中同時掌握總經、個股動態、技術籌碼與決策沙盤，V4.0 採用全自適應雙欄/三區塊黃金比例佈局：
* **頂部宏觀與熱錢指南針 (`Macro & Forex Strip`)**：
  * 於畫面最上方橫幅即時推播國際核心指數（道瓊、納斯達克、標普500、費城半導體期貨）、外匯與原物料熱錢走向（美元指數 DXY、新台幣匯率、布蘭特原油、黃金現貨）以及亞股連動快照（日經225、恆生、上證指數）。
  * 字體升級至自適應 `text-xs sm:text-sm` 飽滿比例，一目瞭然掌握大盤多空風向。
* **左欄：個股報價、20 日動態黃線防守與五檔深度 (`Left Command Panel`)**：
  * 精簡收斂至 `max-w-[360px]`，支援個股即時現價、20 日動態黃線回撤防守與安全邊際底線、五檔委買賣深度推演與成交量分佈。
  * 內建防跳行 (`whitespace-nowrap & truncate`) 與彈性字級校準，視窗縮放時杜絕標題與價格擠壓折行。
  * 下方整合 **即時監控清單 (`Watchlist`)**，隨時點選 `00919`、`2330`、`2454` 等標的，全頁面資料立刻動態連動切換。
* **中央：6 大 Subplots 多週期切換主圖 (`Central ECharts Engine`)**：
  * 自適應彈性擴張 (`flex-1 min-w-0`)，徹底消除水平邊界裁切與溢出。
  * 支援 **「日K (短線)」與「週K (波段)」多週期即時重採樣切換**，單圖整合主圖 K 線/均線/波段防守線、成交量能、MACD 趨勢、KD 動能、RSI 強弱與籌碼追蹤 6 大副圖。
* **右欄：5:7 雙對照智能決策矩陣 (`Dual-Pane Intelligence Grid`)**：
  * 在超寬螢幕與標準解析度下自動切分為雙子區塊：
    * **【右左區塊 col-span-5】**：上方呈現 **4 大市場維度快覽** (Pillar 1 國際費半/科技龍頭、Pillar 2 ADR 溢價與期貨夜盤、Pillar 3 內資法人主力、Pillar 4 散戶融資心理)；中間附帶專業分析師近期數據佐證；下方緊接 **個股即時輿情與重大快訊卡**，畫面緊湊無空白。
    * **【右右區塊 col-span-7】**：獨佔寬裕空間，呈現 **CIO 終極決策指令與策略依據**，下方掛載 **Wargame Council 10 輪多空對抗沙盤推演會議完整紀錄**，且預設全展開到底 (`flex-grow min-h-[420px]`)，完整揭露四大派系交鋒脈絡。

### 2. Wargame Council 10 輪對抗沙盤推演系統 (`AI Multi-Persona Debate Engine`)
V4.0 首創個股專屬多空沙盤推演會議，由系統模擬 4 位頂級分析師針對當前選定個股 (`targetSymbol`) 展開 10 輪激辯，並由 CIO 統整給出最終策略指引：
1. **高盛量化派 (Goldman Quant)**：專攻統計套利、期貨期現貨價差與波動率模型。
2. **國泰投信基本面 (Cathay Fundamental)**：聚焦 EPS 成長動能、產業趨勢、殖利率防禦力與估值區間。
3. **統一投顧籌碼主力 (President Technical)**：解構外資與投信進出連續性、主力分點吃貨與散戶融資籌碼清洗度。
4. **群益長線防守 (Capital Defensive)**：著重安全邊際、下檔支撐韌性、最大回撤風險與尾部避險策略。

### 3. 200 萬信貸槓桿與月月配現金流對衝儀表板 (`Analytics & Cashflow Engine`)
* **信貸與利息差額對衝**：單一介面試算銀行月均攤還金額 (`27,044 元`) 與 100 萬月月配 ETF 預估利息 (`8,419 元`) 之差額對衝，實繳減壓達 31.1%。
* **「以息養股」零股複利試算器**：動態模擬每月領取配息後投入 `2330 台積電` 或 `2454 聯發科` 零股累積之複利曲線，兼顧穩定防守與超額報酬。

## 📱 三、 跨裝置全平台適配與 PWA 安裝指南

本系統採用純前端架構 (`HTML5 + Tailwind CSS + ES6 JS`)，無須安裝繁雜的 Python / Node 後端即可開箱即用。
* **電腦端 (`Desktop / Ultrawide`)**：推薦以解析度 `1600x1000` 以上或超寬螢幕 `3440x1313` 瀏覽，完美體驗 5:7 雙對照全景戰情室。
* **手機端 (`Mobile PWA`)**：
  1. 使用 iOS **Safari** 或 Android **Chrome** 開啟系統 URL / `index.html`。
  2. 點擊瀏覽器分享工具列 → 選擇「**加入主畫面 (Add to Home Screen)**」。
  3. 桌面即會產生專屬 App 圖標，點擊即可開啟離線支援與全螢幕沉浸式行動看盤介面。
